#### New user features tour

A new interactive checklist helps you learn the key features of Table Slayer. New users will see the checklist automatically in the right panel, guiding them through essential features like setting up their TV size, using fog of war, placing markers and lights, and more. All users can access the tour anytime by clicking the "Learn" button in the bottom right corner of the stage. [#434](https://github.com/Siege-Perilous/tableslayer/pull/434)
