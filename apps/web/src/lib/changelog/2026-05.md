#### New user features tour

Table Slayer now includes a feature tour checklist that guides users through features like setting up their TV size, using fog of war, placing markers and lights, and more. All users can access the tour anytime by clicking the "Learn" button in the bottom right corner of the stage. [#434](https://github.com/Siege-Perilous/tableslayer/pull/434)
